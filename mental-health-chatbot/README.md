# Mental Health Chatbot

This project is an AI-powered mental health chatbot that provides support and resources for users.

## Project Structure
- **frontend/**: Contains the React application.
- **backend/**: Contains the Node.js server.

## Getting Started

### Frontend
1. Navigate to the `frontend` directory.
2. Run `npm install` to install dependencies.
3. Run `npm start` to start the React application.

### Backend
1. Navigate to the `backend` directory.
2. Run `npm install express` to install Express.
3. Run `node server.js` to start the server.

## Features
- Real-time chat with the AI chatbot.
- Resource library for mental health articles and videos.
- Mood tracking functionality.
- Emergency contact information access.
